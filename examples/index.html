üöÄ Quick Example

Here's how easy it is to create a rich text box:

import { RichText } from './components'; const richText = new RichText({ x: 100, y: 100, width: 300, height: 200, text: 'Hello World!', style: { fontSize: 24, fontWeight: 'bold', color: '#FF6B6B' }, draggable: true }); layer.add(richText);
üí° Pro Tip: The RichText component works just like native Konva shapes. You can use all standard Konva properties like x, y, scaleX, scaleY, rotation, and draggable.

üéØ Key Features

  • Character-Level Styling - Apply different styles to individual characters
  • Full Keyboard Support - Ctrl+B for bold, Ctrl+I for italic, and more
  • Undo/Redo - Complete history management with Ctrl+Z and Ctrl+Y
  • Copy/Paste - Preserves styles when copying and pasting
  • Lists - Bullet and numbered lists with automatic nesting
  • Transformations - Scale, rotate, and transform like any Konva shape
  • Event System - Listen to editstart, editend, and textchange events
  • JSON Serialization - Save and restore text boxes easily
  • TypeScript Support - Fully typed API with IntelliSense support
